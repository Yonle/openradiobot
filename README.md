## OpenradioBot
A telegram bot that uses [openradio](https://github.com/Yonle/openradio) and [ytdl-core](https://npmjs.com/ytdl-core) to create a radio.

## Configuration
Create `.env` files, And fill:
```
# Fill your bot token here....
BOT_TOKEN=
```
